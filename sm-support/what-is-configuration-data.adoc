= 構成データとは何ですか？
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Collect Configuration Dataを選択すると、RAID構成データベースの現在の状態が保存されます。

RAID構成データベースには、コントローラ上のボリュームグループとディスクプールに関するすべてのデータが含まれています。構成データの収集機能では、のCLIコマンドと同じ情報が保存されます。 `save storageArray dbmDatabase`。
